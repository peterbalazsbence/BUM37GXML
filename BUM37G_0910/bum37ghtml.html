Git parancsok listája

Beállítások

git config --global user.name "Név"
Beállítja a nevedet
git config --global user.email "email@example.com"
Beállítja az email címedet

Új projekt

git init
Új git mappát csinál
git clone [link]
Lemásol egy projektet a gépedre

Alap parancsok

git status
Megmutatja mi változott
git add [fájl neve]
Hozzáad egy fájlt a mentéshez
git add .
Az összes változott fájlt hozzáadja
git commit -m "üzenet"
Elmenti a változásokat üzenettel

Történet megnézése

git log
Megmutatja a mentések listáját
git log --oneline
Rövidebb lista a mentésekről
git diff
Megmutatja pontosan mi változott

Branch-ek

git branch
Kilistázza az ágakat
git branch [új név]
Új ágat csinál
git checkout [ág neve]
Átvált másik ágra
git merge [ág neve]
Összekeveri két ágat
